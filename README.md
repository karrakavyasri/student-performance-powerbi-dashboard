# 🎓 Student Performance Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes student performance data using Power BI to gain insights into:
- Grade-wise student distribution
- Pass & fail analysis
- Department-wise performance
- Engagement and academic outcomes

## 🛠 Tools Used
- Power BI
- Microsoft Excel / CSV
- Data Visualization techniques

## 📊 Dashboard Features
- KPI cards for average score and pass rate
- Grade-wise student count
- Result category comparison (Pass / Fail / Average)
- Parent education vs student performance analysis

## 📁 Files Included
- `Student_Performance_Dashboard.pbix` – Power BI report file
- `screenshots/` – Dashboard preview images
- `dataset/` – Student dataset used for analysis
